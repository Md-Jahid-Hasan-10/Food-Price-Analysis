# Food Price Analysis 📊

This project analyzes global food price trends using **Python**.  
It explores the relationship between currency strength (USD, JPY) and food inflation and provides data-driven insights using visualizations.

---

## 🛠️ Technologies Used

- **Python 3**
- **Pandas** – Data cleaning and transformation
- **NumPy** – Numerical computations and statistical analysis
- **Matplotlib** – Data visualization and trend plotting

---

## 🔍 Key Insights

- **Price Distribution:** The most frequent average price observed in the dataset is **4.03**.
- **Currency Impact:** Food prices tend to rise when the local currency is weak against the USD.
- **Outliers:** Successfully identified and handled data surges caused by high-value currencies like the Japanese Yen.

---

## 📊 Project Workflow

1. **Data Loading:** Load CSV or Excel dataset containing global food prices.
2. **Data Cleaning:** Handle missing values, normalize currencies, and remove outliers.
3. **Exploratory Data Analysis (EDA):** Visualize trends, distributions, and correlations.
4. **Analysis:** Explore how currency fluctuations impact food prices.
5. **Visualization:** Plot trends, distributions, and comparisons using Matplotlib.

---

## 📂 How to Use

1. Clone this repository:

```bash
git clone https://github.com/Md-Jahid-Hasan-10/Food-Price-Analysis.git
```

2. Navigate to the project folder:

```bash
cd food-price-analysis
```

3. Open the Jupyter Notebook file (`.ipynb`) in **Jupyter Notebook** or **Google Colab**:

```bash
jupyter notebook Food_Price_Analysis.ipynb
```

4. Run the cells to explore the analysis and visualizations.
